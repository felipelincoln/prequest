    live_modal(socket, component, opts)

    View Source

    Renders a component inside the PrequestWeb.ModalComponent component.

    The rendered modal receives a :return_to option to properly update the URL when the modal is closed.

    Examples 

    <%= live_modal @socket, PrequestWeb.UserLive.FormComponent,
  id: @user.id || :new,
  action: @live_action,
  user: @user,
  return_to: Routes.user_index_path(@socket, :index) %>
