    Prequest v0.1.0 Prequest.Accounts.User View Source

    User is the schema in the Accounts context that model the users.

    It consists of a :bio, :name and :username field plus the timestamps().

    schema "users" do
  field :bio, :string
  field :name, :string
  field :username, :string

  timestamps()
end
